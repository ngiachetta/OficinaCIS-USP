Xpath

O que é Xpath?

O Xpath é linguagem que nós podemos usar para extrair "query set" de arquivos XML.

Podemos pegar os nodes com expressões Xpath

Um exemplo disso é //tag[@attr=value], que significa: "Encontre todas as tags com o nome tag, cujo atributo é atributo e possui como valor, value"

Também podemos especificar o caminho até a tag de interesse
